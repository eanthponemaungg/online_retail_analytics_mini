# Online Retail Analytics Mini-Dashboard (Updated!)
An interactive Python mini-dashboard using Streamlit and Pandas for retail data tracking, management and dynamic sales visualization with filters.

## Features
### Role-Based Access
*   **Admin**: Full CRUD operations (Add, Update, Delete records), filter data, and export to Excel.
*   **User**: View-only access to interactive analytics and charts.

### Dynamic Analytics
*   Multi-filtered charts by date range and country.
*   Interactive Sales, Quantity, Scatter, and Time-Series visualizations.

## Tech Stack
*   **Language**: Python
*   **Framework**: Streamlit
*   **Data Analysis**: Pandas
*   **Storage**: Excel
